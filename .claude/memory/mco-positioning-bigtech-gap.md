2026-07-10 경쟁 리서치로 갱신 (v0.1 표현: "빅테크가 하기엔 짜치고 사용자는 절실한 빈틈"):

- **재정의**: 빅테크는 '짜쳐서 안 한' 게 아니라 **전부 자기 담장 안에서 구현했다** — ChatGPT 프로젝트 메모리·Dreaming, Claude Managed Agents 스코프+감사로그, Codex 로컬 메모리, Devin Knowledge 게이트. 이것은 수요의 증거다. 그들이 구조적으로 못 하는 것 = **벤더 횡단 중립**(경쟁사에 자사 함대의 기억을 맡길 벤더는 없다).
- **3요소 2026-07 판정**: 크로스 플랫폼 이동성 = 단독 차별점 사망(혼잡) / 계정 바인딩·사용자 소유 = 빅테크 레벨 공백 / **에이전트별 권한 스코프 = 벤더 횡단 완전 공백** ★ (nice-to-have → 필수 인프라 승격).
- 차별점 서술은 항상 **조합 해자**로: 사용자 소유 + 벤더 횡단 권한 스코프 + 정밀 선택 + 게이트 환류 + 브리핑 팩 — 전부 하는 곳 없음(2026-07 확인).
- 앵커: Personal Memory Passport — 유지하되 외부 충돌로 재검토 중 [[mco-term-collision-passport]].

**Why:** '개인화 메모리 모델'로만 말하면 빅테크와 정면 충돌로 읽히고, '짜침' 프레임은 이제 사실과 다르다. '중립이라 못 함' 프레임이 서사와 방어 논리의 핵심.

**How to apply:** 피치·차터·경쟁 분석에서 '벤더 횡단 중립 + 조합 해자' 프레임으로 서술하고, 기능 우선순위도 조합 5요소 기준으로 판단. [[mco-fleet-reframe]] · [[mco-competitive-landscape]] · [[mco-mvp-direction]]
