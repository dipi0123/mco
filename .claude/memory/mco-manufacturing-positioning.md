- 제조업 적용 시 **'LLM 독립형 메모리 레이어' 같은 기술 프레임 금지** — 현장에 와닿지 않는다.
- 정본 포지셔닝: **"현장의 설비·품질·작업자 이력을 기억하고, 지금 상황에 딱 맞는 과거 사례를 꺼내주는 제조 메모리 컨텍스트"** (영문 앵커: Factory Memory Context).
- '지금 상황에 맞는 과거 사례 인출' 라인으로 시작하면 설득력이 훨씬 높다 — [[mco-selection-over-storage]]의 제조 버전.

**Why:** 제조가 유력한 1차 버티컬 후보로 논의됨. 기술 용어가 아니라 현장 가치 언어로 팔아야 설득이 된다.

**How to apply:** 제조 대상 제안·피치 산출물에서 이 표현을 정본으로 사용. 다른 버티컬 확장 시에도 같은 패턴(도메인 이력 + 상황 맞는 사례 인출)으로 번역한다.
