2026-07-11 아이디에이션 세션 산출 (페인 증거 3트랙 웹 리서치 기반):

- **페인 서열(실측)**: 오염 > 무영향 > 부재 — 벤더 기억이 보편화되자 파워유저가 기능을 **끄는** 행동이 최상위 증거. "소비자"의 실체 = 컨텍스트가 돈인 프로슈머(빌러블 아워·마감·서류·멀티 법인). 제품 서사: "기억을 준다" → **"기억을 믿을 수 있게 한다"**.
- **발산 14개** (요지): ①마스터 프롬프트 은퇴식 ②법인 프로필 원장→서류 컴포저(멀티 법인 벽 진화) ③회의 배기가스 수확기 ④클라이언트 바이블 오토파일럿 ⑤망각 엔진 ⑥기억 부패도 감사 리포트 ⑦약속 원장 ⑧이력 지갑(의료) ⑨함대 정합 버스(역할③ 진화) ⑩안티패턴 면역계(경험 원장 진화 — 저장→적시 주입) ⑪결정 리콜(인간이 소비자) ⑫채팅 무덤 발굴기(웜스타트) ⑬BYOM 고용 경계 ⑭오프보딩 에스크로.
- **이후 재배치(전략 정렬 후)**: ⑨⑩ = 독립 서비스가 아니라 **슬롯 차별화 기술 자산** / ②⑥ = 풀(pull) **출력 앱**(②=경로 B, ⑥=획득 장치) / ① = T0/T1 설정성 계층으로 흡수 / ⑫ = 첫 5분 웜스타트로 흡수 / ⑪ = 결정 인박스·대시보드의 렌더링 후보.
- **기각·보류 기록(반복 논의 방지)**: 회의 배기가스 수확기 원안 **기각**(Granola가 MCP 출시로 선점) → '회의 **전** 브리핑 팩'으로 반전 재등록(선택>저장 정합) / 개인 GB 저장 구독 **기각**([[mco-viability-verdict]]) / 오프보딩 에스크로 = 직접 증거 부족으로 보류(Engram $98M은 간접 증거일 뿐) / '10배 야심'류(BYOM 고용 경계·이력 지갑)는 북극성으로만 유지.
- **도발에서 나온 신규 논거(차터 후보)**: 함대 게이트의 존재 이유 = 노이즈 차단만이 아니라 **탐색 다양성 보존**(합의된 '결정'만 전파, '가설'은 격리 — 조기 정합은 모드 붕괴 유발).

Why: 발산의 절반은 기각 기록 — 같은 아이디어를 다음 세션에서 재발명하지 않기 위함.

How to apply: 출력 앱·데모 기획은 ②⑥에서, 슬롯 차별화 스펙은 ⑨⑩에서 시작. 미채굴 아이디어(⑤⑦⑪⑬)는 분기 리뷰 때 재평가. [[mco-viability-verdict]] [[mco-architecture-v01]]
