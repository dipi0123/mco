- 'Personal Memory Passport'(2026-07-09 브레인스톰에서 잡은 개념 앵커)는 더 이상 고유하지 않다: **MemoryLake**가 "Memory Passport"를 제품명으로 사용([memorylake.ai](https://www.memorylake.ai/en/products/memory-passport)), **mem0**이 "memory passport"·"Plaid for memory"를 비전 문구로 사용(2025-10 $24M 펀딩 기사).
- **처리 (2026-07-10 합의)**: 정본 용어로 **유지**하되 차터 §9에 재검토 안건 등재. 대외 산출물의 헤드라인·브랜드에는 쓰지 않는다(내부 약칭으로만). 상표 조사 + 서비스 네이밍 논의 시 유지/교체 최종 결정.
- 교체 논의 시 후보 방향(미확정 제안): 여권(사람 소유 정체성) + **'비자'**(에이전트별 스코프·기한 있는 위임) 확장 메타포 — TTL=비자 만료, 출처=도장.

**Why:** 외부 충돌을 모른 채 피치에 쓰면 "이미 있는 것"으로 읽힌다. 기각/재론 방지용 기록.

**How to apply:** 대외 산출물에서 이 용어를 전면에 세우지 않는다. 네이밍 확정 시 이 메모리와 차터 §2·§10을 함께 갱신. [[mco-competitive-landscape]] · [[mco-positioning-bigtech-gap]]
