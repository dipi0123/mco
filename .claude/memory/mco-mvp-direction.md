- **MVP 구성(구상):** ① **MCP 메모리 서버** — 어떤 LLM이든 연동하는 코어 표면 ② **브라우저 확장** ③ **대시보드** — 내 메모리를 보고 관리하는 UI.
- **초기 타깃:** 파워유저 — 개발자·컨설턴트·변호사처럼 여러 LLM/에이전트를 오가는 직군.
- 인증/소유: OAuth 계정 바인딩 — [[mco-positioning-bigtech-gap]].

**Why:** 여러 LLM/에이전트를 오가는 직군이 크로스 플랫폼 맥락 단절 페인을 가장 절실히 느낀다 — MVP 검증의 최단 경로.

**How to apply:** MVP 스펙 작업 시 이 3요소에서 출발. 구성·타깃이 바뀌면 이 메모리를 갱신. [[mco-concept-core]].
